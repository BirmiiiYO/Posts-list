# You can:
 - View posts that are on the server;
 - Add new post;
 - Edit existing posts;
 - Like posts;
 - Sort by likes;
 - Sort by name;
 - Sort by title;
 - Search by keywords;
 - Choose the number of elements per page and the page itself;

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
